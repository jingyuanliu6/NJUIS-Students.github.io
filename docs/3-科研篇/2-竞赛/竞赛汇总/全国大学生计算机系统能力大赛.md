# 全国大学生计算机系统能力大赛

## 竞赛简介

全国大学生计算机系统能力大赛（含龙芯杯等赛道）由教育部高等学校计算机类专业教学指导委员会主办，涵盖 CPU 设计、编译系统、操作系统、数据库系统等赛道，旨在培养我国高端芯片及关键基础软件的后备人才。

## 参赛要求

- **参赛对象**：普通高等学校全日制在校本科生和专科生
- **赛道设置**：
  - **CPU设计赛（龙芯杯）**：分为团体赛和个人赛
  - **编译系统设计赛（华为毕昇杯）**：分编译系统设计赛道和编译系统挑战赛道
  - **操作系统设计赛**：分内核实现赛道和功能挑战赛道
  - **数据库管理系统设计赛**
- **组队方式**：每队1-4人，来自不同学校的学生不能联合组队
- **报名费用**：大赛不收取任何报名费、培训费、评审费等费用

## 时间安排

- **报名时间**：每年3月-6月
- **初赛作品提交**：每年8月5日前
- **初赛评审**：每年8月
- **全国总决赛**：每年8月中下旬
