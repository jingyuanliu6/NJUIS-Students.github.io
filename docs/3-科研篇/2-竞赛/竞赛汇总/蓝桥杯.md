# 蓝桥杯

## 竞赛简介

蓝桥杯全国软件和信息技术专业人才大赛是工业和信息化部人才交流中心举办的全国性 IT 学科赛事，涵盖程序设计、电子设计等多个方向。

## 参赛要求

- **参赛对象**：具有正式全日制学籍且符合相关科目报名要求的研究生、本科生及高职高专学生
- **赛项设置**：
  - **软件赛（个人赛）**：C/C++程序设计、Java软件开发、Python程序设计、Web应用开发、网络安全
  - **电子赛（个人赛）**：嵌入式设计与开发、单片机设计与开发、EDA设计与开发、物联网设计与开发、FPGA设计与开发、5G全网规划与建设
  - **项目实战赛（个人赛）**：人工智能、智能体开发
  - **数字科技创新赛（团队赛）**
  - **视觉艺术设计赛**
  - **专项赛**
- **组别划分**：研究生组、大学A组、B组、C组，职业院校组

## 时间安排

- **报名时间**：每年10月至12月
- **省赛/全国选拔赛**：次年4月
- **全国总决赛**：次年6月中上旬
