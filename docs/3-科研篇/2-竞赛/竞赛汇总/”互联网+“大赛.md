# “互联网+”大赛（中国国际大学生创新大赛）

## 竞赛简介

中国国际大学生创新大赛（原“互联网+”大学生创新创业大赛）是由教育部等部门主办的全国性创新创业竞赛，是目前国内最具影响力的大学生双创赛事。

## 参赛要求

- **参赛对象**：在校或毕业5年内的中职中专、高职高专、职业教育本科、普通本科、博硕士研究生均可参加
- **组队方式**：以团队为单位报名参赛，每个团队3-15人（含团队负责人），须为项目实际核心成员
- **赛道设置**：
  - **高教主赛道**：分本科生组和研究生组，各设创意组、创业组
  - **青年红色筑梦之旅赛道**
  - **职教赛道**
  - **产业赛道**：企业命题组、成果转化组
  - **萌芽赛道**
- **项目类别**：新工科类、新医科类、新农科类、新文科类、人工智能+、低空经济、生物技术、量子科技、新能源、新材料等

## 时间安排

- **大赛启动**：每年4月
- **校级初赛**：4-6月
- **省级复赛**：4-8月
- **全国总决赛**：9月
- **报名截止**：通常为9月初
