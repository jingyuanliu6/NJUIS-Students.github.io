# 全国大学生数学建模竞赛

## 竞赛简介

全国大学生数学建模竞赛是中国工业与应用数学学会主办的面向全国大学生的群众性科技活动，是全国高校规模最大的基础性学科竞赛。

## 参赛要求

- **参赛对象**：所有全日制在校大学生
- **组队方式**：每队不超过3人，专业不限
- **竞赛组别**：
  - 本科组：所有大学生均可参赛
  - 高职高专组：仅限高职高专学生报名（亦可自愿选择参加本科组）
- **重要规则**：
  - 竞赛期间参赛队员必须独立完成赛题解答
  - 不得与他人（包括指导教师）交流讨论赛题问题
  - 可使用图书资料、计算机和软件
  - 可使用AI工具作为辅助，但须对作品原创性负责

## 时间安排

- **报名时间**：每年4月至9月初
- **竞赛时间**：每年9月第一个周四18:00至周日20:00（72小时）
- **截止报名**：每年9月1日20:00
- **交卷时间**：每年9月7日20:00
