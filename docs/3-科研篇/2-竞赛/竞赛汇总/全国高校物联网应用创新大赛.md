# 全国高校物联网应用创新大赛

## 竞赛简介

全国高校物联网应用创新大赛由教育部科技发展中心与互联网应用创新开放平台联盟联合主办，面向全国高校物联网相关专业学生，旨在激发学生创新创业热情，推动物联网技术应用。

## 参赛要求

- **参赛对象**：普通高等院校、高职院校、二级学院、独立学院、本、专科在校大学生及研究生均可报名参加，专业不限
- **组队方式**：以学校为单位组队参赛，每队不超过3名队员及1名指导教师
- **赛道设置**：
  - 创意赛：线上竞赛和分区答辩相结合
  - 挑战赛：采用线上竞赛方式
  - 命题赛：不设预赛

## 时间安排

- **报名时间**：每年7月-10月（具体以当年通知为准）
- **预赛阶段**：每年4-6月各赛区进行分区决赛
- **全国总决赛**：每年10月左右
