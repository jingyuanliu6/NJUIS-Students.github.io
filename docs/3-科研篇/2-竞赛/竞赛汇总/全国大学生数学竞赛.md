# 全国大学生数学竞赛

## 竞赛简介

全国大学生数学竞赛（CMC）是中国数学会主办的面向本科生的全国性高水平学科竞赛，旨在培养大学生的数学思维能力。

## 参赛要求

- **参赛对象**：所有全日制在校大学生
- **竞赛类别**：
  - **数学专业类**：数学类专业（代码0701）学生只能报考数学专业类
  - **非数学专业类**：非数学类专业的学生报考类别不限
- **组别细分**：
  - 数学专业类分A类和B类
  - 非数学专业类分A类（理工类）和B类（经管文史类）
- **报名费用**：约100元/人（以当年通知为准）

## 时间安排

- **初赛时间**：每年11月（具体日期另行通知）
- **初赛考试时长**：2.5小时（9:00-11:30）
- **决赛时间**：次年3月或4月
- **报名截止**：每年10月中旬
