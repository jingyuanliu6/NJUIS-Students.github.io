# 研究方向导航：先看看都有哪些路

很多同学一提到「科研方向」只想到模糊的几个词：AI、系统、网络、密码学…… 但具体在做什么、差别在哪里，其实很难一眼看懂。

这篇文档的核心就是下面这张**交互式地图**，你可以在里面拖动、缩放、点开感兴趣的节点，快速浏览各个方向的大致全景。

## 一、怎么用这张图？

简单建议一个使用方式：

- 先整体缩小，看一眼有哪些「大块」：比如系统、安全、AI、图形学、人机交互、理论等；
- 对自己有好感的块，放大进去看看有哪些更细的子方向；
- 记下你觉得「名字听起来很对胃口」的几个方向名字，作为后续查资料/找导师的关键词。

不需要现在就完全搞懂每一个词的含义，先知道「世界上原来有这么多种玩法」就够了。

---

## 二、如果我对某个方向感兴趣，下一步可以做什么？

你可以尝试下面几件事：

- 用中文/英文搜索这个方向的简介（例如「操作系统研究方向」「computer systems research」）；
- 去对应方向的老师主页看看他们最近在做什么项目；
- 在《细分方向经验分享》里看看有没有相关方向的学长学姐故事；
- 如果已经下定决心想做科研，可以带着这些方向关键词，去看《如何从联系导师到做出一篇论文》。

---

## 三、这张图只是起点

科研方向本身在不断变化，新方向在涌现，旧方向在交叉、融合。

这张图更像是一张「老地图」：帮你快速建立一个大概的认知，但不要被它框死——真正有意思的地方，往往在边界和交叉的地方。
