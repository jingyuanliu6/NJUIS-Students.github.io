# 海外留学经验分享

相信大家都有一颗我想要"出去看看"的心，当然了，大多数同学都会停留在"想"这个阶段，因为跳出固有体系是一件很烦的事情，需要额外做很多努力，并且海外欧美体系下的申请，和国内差距非常大。这里，我主要分为海外硕士和海外博士两块进行介绍，两者最大的区别是前者是需要支付一笔不少学费的，后者则更加像一份工作，尤其是现在很多人都会叫自己的博士导师"老板"，海外博士是更贴切的，因为生活费实打实来自老板🥵。

除了按照硕士和博士这个维度进行区分之外，主要的分类，我选择的是地区，主要focus在我了解的两大地区：美国，港新。其余可能的留学地方可能是欧洲和澳洲，但是我了解太少，此处就不进行介绍。

## 海外硕士

本项目适合，家里愿意提供一年40万人民币（至少）经费支持，自己想体验国外生活或者想更快拿到硕士学位，开始就业的同学。

### 我眼中的海外硕士

互联网有一个热词叫"水硕"，但是并非所有硕士，都是水的。

事实上，"水不水"取决于两个因素：**项目**和**投入程度**。

先说项目本身。海外硕士大致可以分为两类：**授课型（Taught/Coursework）** 和 **研究型（Research/Thesis-based）**。授课型硕士通常1-1.5年，以上课为主，适合想快速提升学历、转换赛道或为就业做准备的同学。研究型硕士则更接近博士的前期训练，需要跟导师做研究、写thesis，时长通常2年，适合未来有读博打算的同学（实际上研究型硕士基本就可以视作博士）。

大家口中的"水硕"，往往指的是部分授课型项目——课程强度低、录取门槛不高、一年就能毕业。但即使是授课型硕士，顶尖项目的含金量依然很高。比如CMU的MSCS、MSML，Stanford的MSCS，课程压力和项目难度都不低，并且这层名校的背景，往往是申博的同学很难拿到的。

再说个人投入。同一个项目里，有人天天泡实验室、找实习、发论文；也有人混完学分就走。所以"水不水"很大程度上取决于你自己怎么利用这段时间。海外硕士最大的价值其实不只是那个学位，而是你能接触到的资源——顶级教授、industry connection、career fair、以及一个完全不同的学术和文化环境。

**按地区来看：**

**美国硕士：** 美硕是最主流的选择，尤其是CS、EE等理工科方向。项目时长1-2年不等，学费每年大约30-60万人民币（取决于公立/私立），加上生活费，总预算大约在60-120万人民币。美硕最大的优势是就业资源：OPT（毕业后工作签证）给你1-3年的时间在美国找工作，而且美国的科技公司薪资水平远高于国内。劣势是贵，以及近年来H1B签证的不确定性。（实际我在国内的公司已经看到不少美硕了）

**港新硕士（香港 & 新加坡）：** 港新硕士是性价比极高的选择。学费大约10-25万人民币/年，生活费也比美国低不少。项目时长通常1-1.5年。港三（港大、港中文、港科技）和新二（NUS、NTU）在世界排名上非常能打，QS排名甚至比很多美国学校更好看（如果你在意这个的话，进国企等是非常重要的指标）。劣势是硕士项目规模较大、部分项目偏"授课流水线"，以及港新的就业市场体量远不如美国，最后大多数人就业和实习都会选择国内，所以实习这点实习上面，香港优势比新加坡更大。

### 如何申请

海外硕士的申请流程和国内考研完全不同——没有统一考试，采用的是**材料申请制**。你需要准备一套申请材料，提交给每所学校，由招生委员会综合评估。

**核心申请材料：**

1. **GPA / 成绩单：** 这是最重要的硬指标。美国top项目通常期望GPA 3.7+/4.0（大约对应国内均分88+），港新top项目大约85+。GPA不够的话，其他方面需要非常突出才能弥补。

2. **标准化考试：**
   - **托福/雅思：** 语言成绩是门槛，托福100+或雅思7.0+是比较安全的线。
   - **GRE：** 美国部分项目要求或建议提交GRE（点名CMU），建议325+。港新大部分项目不强制要求GRE，但有的话是加分项。

3. **简历（CV）：** 突出你的研究经历、实习经历、项目经历和技能。理工科申请中，research experience和industry internship是非常加分的。

4. **个人陈述（Statement of Purpose / Personal Statement）：** 解释你为什么要申请这个项目、你的背景如何与项目匹配、未来的职业/学术规划。这是你在"纸面"上展现个性和动机的唯一机会，务必认真打磨。

5. **推荐信（2-3封）：** 找了解你的教授或实习主管写。推荐信的质量很重要——一封来自知名教授的强推可以显著提升你的竞争力。

**申请时间线（以秋季入学为例）：**

| 时间 | 事项 |
|------|------|
| 大三上（9-12月） | 准备托福/雅思，开始了解项目 |
| 大三下（1-6月） | 考GRE（如需要），积累研究/实习经历 |
| 大四上（9-12月） | 撰写文书、准备申请材料、陆续提交申请 |
| 大四下（1-4月） | 收到offer，做决定 |

**Tips：**
- 关于中介：中介我觉得是可以找的，而且找的话一定要越早越好，因为价格都一样，越早找，对整个过程的帮助越大。但是不要过度依赖中介。中介可以帮你处理流程性的事务，但文书一定要自己主导，因为没人比你更了解自己。
- 多看目标项目的官网和在读学生的分享，了解项目的真实风格。
- 申请是一个"组合拳"：GPA是基础，科研/实习是亮点，文书是讲故事的能力，推荐信是第三方背书，缺一不可。

### 优缺点分析

**优点：**

- **时间成本低：** 1-1.5年即可毕业（对比国内学硕3年），更快进入就业市场。
- **全球视野：** 接触不同的学术文化、思维方式和人脉网络，这种体验是无法被替代的。
- **就业竞争力：** 尤其是美硕，毕业后有机会直接在美国就业，薪资和职业起点通常较高。港新硕士回国就业在简历筛选上也有一定优势。
- **灵活性：** 可以作为读博前的跳板（尤其是本科背景不够强的同学），也可以直接就业。

**缺点：**

- **经济压力：** 美硕总花费60-80万人民币，港新也需要30-50万。对大多数家庭来说这不是一笔小数目。
- **不确定性：** 美国的签证政策、就业市场波动都会影响。
- **含金量参差不齐：** 需要自己做好功课，选对项目。盲目追求学校排名而忽略项目质量是常见的坑。
- **适应成本：** 语言、文化、生活方式的差异，对一些同学来说需要时间适应。

## 海外博士

### 我眼中的海外博士

读博是一个非常严肃的人生决定，它不是"硕士的延长版"，而是一种完全不同的生活方式。海外硕士我的建议是：如果你家里愿意支付这笔消费，这是人生再难有的机会；海外博士，我的建议是如果对科研没有足够的坚持，不要来。

当然先说最现实的部分：**钱**。美国和港新的博士项目通常都提供全额奖学金（tuition waiver + stipend）。美国PhD的stipend大约每月2000-3500美金（取决于学校和地区），港新的博士奖学金大约每月1.5-2万港币/2500新币。虽然不算富裕，但基本可以覆盖日常生活开销，不需要家里额外资助。所以博士确实更像一份"工作"——你用时间和脑力换取报酬。

再说学术层面：博士的核心任务是**做出原创性的学术贡献**。这意味着你需要在4-6年（美国）或3-5年（港新）的时间里，深入一个细分方向，发现新问题、提出新方法、发表论文、最终写出一篇博士论文。这个过程充满了不确定性——实验可能反复失败，论文可能被反复拒稿，研究方向可能需要中途调整。

**读博最重要的不是你有多聪明，而是你有多能忍受不确定性和挫折。**

导师（advisor）的选择比学校排名更重要。这点只有自己读了博士之后才有体会，你的资源不再像本科那样直接来自学校/学院，而是直接来自于你的导师。一个好的导师会在学术上给你指引、在职业发展上给你支持、在心理上给你空间。一个不合适的导师则可能让你的博士生涯变成噩梦。所以，选导师一定要慎之又慎——看他的论文风格、问他现在和之前的学生、了解他的指导方式和毕业学生的去向。

**按地区来看：**

**美国PhD：** 美国博士体系最大的特点是**时间长但训练扎实**。通常5-6年，前1-2年上课+rotation（轮转选导师），之后进入正式的研究阶段。美国的优势在于学术资源极其丰富——顶级会议、industry合作、visiting机会多。毕业后无论是走学术界（教职）还是工业界（research scientist），美国PhD都是最被认可的。

**港新PhD：** 港新博士的优势在于性价比和效率。学制通常4年，没有美国那么多的课程要求，更快进入研究状态。港三和新二的CS、EE等方向近年来发展非常快，不少课题组的产出质量已经可以和美国top 20抗衡。另一个优势是地理位置——离国内近，文化适应成本低，回国就业也方便。劣势是学术圈的影响力和network整体上还是不如美国，还有一些信息可以看我写的这个[知乎回答](https://www.zhihu.com/question/464474706/answer/1922733688392750311)。

**申请难度：** 需要注意的是AI领域现在博士的申请难度已经严重超标，尤其是其中的CV，MLLM，NLP，LLM，Agent等细分方向；同学们，如果实在不行可以去一些相对交叉AI的方向，其申请难度会比直接申请AI的小不少。很多导师的PhD位置，都有五人以上在排队等，尤其是热门导师。

### 如何申请

博士申请和硕士申请有一个本质区别：**硕士申请看的是你的硬指标（GPA、经历），博士申请看的是你的导师是否想要你（研究潜力，强com制当然也是要看背景，但是导师要你的话，还是能捞起来）。**

**核心申请材料：**

1. **研究经历：** 这是博士申请中最重要的部分。招生委员会和导师想看到的是：你是否有能力独立开展研究？你是否在某个方向上有足够的深度？发表过论文（哪怕是workshop paper或preprint）是非常强的信号，但没有论文也不是致命的——重要的是你能展示出清晰的研究思路和解决问题的能力。

2. **推荐信：** 博士申请中推荐信的权重极高，甚至可能是决定性因素。最理想的情况是有一封来自你的研究导师的强推，详细描述你在研究中的具体贡献和能力。如果推荐人和目标导师认识，那就更好了——学术圈很小，人脉网络真的很重要。
   
3. **实际情况：** 实际情况是，对于现在的申请来说，首先看的是connection，这个导师是不是和你合作了足够久，或者他很好的朋友和你合作了很久，那么，只有有一个很被他认可的项目，你就能被收下读博；如果没有很直接的connection，那么就是看你paper的情况了，这里注意，一篇对口的，相对有影响力的paper其实就够了，很多本科发表了过多paper的，会over-qualified，比较强的同学，都会是一篇入门作+一篇很solid的作品，吸引到目标导师，这是最好的。

4. **个人陈述 / 研究计划：** 需要清晰地阐述你的研究兴趣、过去的研究经历如何引导你走到现在、以及你未来想做什么。和硕士文书不同的是，博士的SOP需要展示你对目标研究方向的理解深度。

5. **GPA & 标准化考试：** 依然重要，但相对硕士申请，权重会降低。GPA 3.5+/4.0是比较安全的线。GRE同理，有些项目已经不要求了。

6. **套磁（Contacting Professors）：** 这是博士申请中非常关键但常被忽视的环节。在申请前（通常9-10月），主动给你感兴趣的导师发邮件，介绍自己的背景和研究兴趣，表达合作意向。套磁的目的不是让导师当场答应你，而是让他在审材料时记住你。一封好的套磁邮件应该简洁、具体、展示你对导师工作的了解。

**申请时间线（以秋季入学为例）：**

| 时间 | 事项 |
|------|------|
| 大三暑假 | 积累研究经历（暑研、实验室项目等） |
| 大四上（9-10月） | 开始套磁，确定目标导师和学校 |
| 大四上（10-12月） | 撰写文书，提交申请（美国大部分DDL在12月中旬） |
| 大四下（1-3月） | 面试（部分学校/导师会安排面试） |
| 大四下（3-4月） | 收到offer，做最终决定（美国统一的decision deadline是4月15日） |

**Tips：**
- 暑期研究（summer research）是博士申请的利器。如果有机会去目标学校做暑研，既能获得research经历，又能拿到强推荐信，还能提前了解导师和实验室的风格。
- 博士申请是一个"匹配"过程，不仅仅是你被学校选择，也是你在选择导师和环境。面试的时候大胆问导师问题：funding情况、毕业年限预期、学生的日常工作方式等。
- 不要只看学校排名。一个在某个细分领域顶尖的课题组，即使学校综合排名不那么高，也可能比名校里一个普通课题组更有利于你的发展。

### 优缺点分析

**优点：**

- **经济负担小：** 全奖覆盖学费和生活费，不需要家里资助（甚至还能攒点钱）。
- **深度学术训练：** 博士是目前获得系统性研究训练的最佳途径。你会学到如何定义问题、设计实验、批判性思考、撰写论文和进行学术交流。
- **职业天花板高：** 博士学位在学术界是必需的，在工业界的高端研究岗位（research scientist等）也是强加分项。
- **个人成长：** 读博是一段深刻的自我探索过程。你会被迫面对自己的极限、学会和不确定性共处、培养出极强的独立思考和解决问题的能力。

**缺点：**

- **时间成本高：** 4-6年不是一个短的时间。你的同龄人可能已经在工作、赚钱、结婚，而你还在实验室里debug代码。
- **心理压力大：** 博士期间的心理健康问题非常普遍。孤独感、冒名顶替综合症（imposter syndrome）、和导师的关系问题、论文被拒的挫败感——这些都是真实存在的挑战。
- **机会成本：** 尤其对CS等就业市场火热的专业，硕士毕业直接就业的收入可能在短期内远超博士的stipend。
- **不确定性：** 研究本身就是不确定的，你无法保证自己一定能在预期时间内毕业，也无法保证毕业后能找到理想的学术职位。
- **导师风险：** 导师的风格、资金状况、甚至人品都会直接影响你的博士体验。选错导师的代价可能非常大。

---

> **最后想说的话：** 无论是硕士还是博士，出国留学从来不是一条"必须走"的路，而是一个"适不适合自己"的选择。想清楚目标、评估好经济状况和心理承受能力，再做决定。一旦决定了，就全力以赴——这段经历终将成为你人生中最独特、最有价值的篇章。
另外，我想给所有大学生一个建议，包括立志读博的同学：尽量找一段工业界的实习。不要一直待在学校的"象牙塔"里。走出去你会发现，这个世界的可能性远比你想象的多——OpenAI、Anthropic 等顶尖公司有大量本科学历的核心员工，理想汽车的创始人李想甚至只有高中学历。学历是路径，不是天花板。有一段海外经历也非常重要，你的信息源会得到极大的扩展，不止局限在简中世界。
