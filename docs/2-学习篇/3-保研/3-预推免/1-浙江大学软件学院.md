# 2025浙大软院预推免

## 基本信息
浙大软件学院硕士包含人工智能和关键软件两个方向。2025预推免入营800+人，机试参与人数600人左右，录取200人。考核分为机试和面试，面试占比85%，所以主要还是看面试情况。最后录取候补到400+位次，成功概率还是很大的。

具体安排如下：
- 9月16日9:00个人陈述提交截止：每位申请人需准备5-8分钟的陈述PPT（PDF格式），必须包含本科成绩单，全面自述呈现英语能力、专业能力和研究潜力。
- 9月17日18:00-21:00上机考试（17：45需要进入考场） 地点：浙大宁波理工学院（宁波市鄞州区钱湖南路1号）石麟大楼 
- 9月18/19日面试（具体时间取决于分组情况） 地点：浙江大学宁波科创中心科研大楼（宁波市鄞州区学府路1号）/宁波市鄞州区学府路5号

## 9月17日 机试
共4道题，时间3小时，ACM风格，只支持c++、Java，每题20-30分，分值随题目难度增加，能反复提交看自己样例过了几个。25年难度比较大，没有人AK，大部分同学的得分集中在三四十分。一般来说，机试得分在五六十分以上会比较有优势，但即使得分在二三十分，只要面试表现好，也有机会通过候补名单上岸。

题目围绕同一背景，重点考察图论、DFS/BFS、二维dp、贪心、前缀和等算法。

## 9月18日 面试
机试后进行，通常持续20分钟。需要准备一个5-8分钟的自我介绍PPT（包含1分钟英文介绍）。面试主要围绕你的PPT、简历和项目经历进行深入提问，考察专业课知识，很关注编程能力。

## 总结
总成绩由机试（15%）和面试（85%）综合决定。由于面试区分度较大，即使机试成绩没有很靠前，只要达到平均水平以上且面试表现突出，总排名就会很高。而且最后放弃的同学会很多，候补到的机会很大。
